# Vagrant Prestashop
A vagrant box for prestashop

## Includes: 
* wget 
* php7.1
* libapache2-mod-php7.1
* php7.1-gd
* mysql-server-5.7
* php7.1-xml
* php7.1-mcrypt
* php7.1-mbstring
* php7.1-mysql
* php7.1-zip
* php7.1-curl
* php7.1-intl
* Prestashop 1.7.1.2

## Prestashop Version
You can install any prestashop version. Edit bootstrap.sh.
